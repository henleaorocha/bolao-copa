# ADR-004: Flag Images via flagcdn.com CDN

## Status

Accepted

## Date

2026-05-24

## Context

The bet modal requires flag images for all 32 Copa 2026 nations. The PRD mandates SVG/PNG flags (not emoji) for rendering consistency across Android and iOS. Two delivery strategies were evaluated.

## Decision

Use `https://flagcdn.com/w80/{iso_alpha2_code}.png` URLs for each nation card. No new npm dependency is introduced. Each flag is a `<img>` element whose `src` points to the CDN. ISO 3166-1 alpha-2 codes are stored in the static `lib/copa-teams.ts` data file alongside team names.

## Alternatives Considered

### Alternative 1: `country-flag-icons` npm package

- **Description**: Install the `country-flag-icons` package. Flags are imported as React SVG components and bundled.
- **Pros**: Works offline and in Jest tests without network mocking. SVG rendering is pixel-perfect.
- **Cons**: Adds ~300KB to the bundle (even after tree-shaking to 32 flags, each SVG component carries overhead). Requires an npm install step.
- **Why rejected**: The app is web-only and always online. Bundle weight and a new dependency are not justified when a CDN URL achieves the same visual result.

## Consequences

### Positive

- Zero npm dependency added.
- Flags load lazily by the browser — no impact on initial JS bundle size.
- Updating a flag (e.g., country redesign) requires no code change.

### Negative

- Requires internet access; flags do not render in offline/PWA mode.
- flagcdn.com is an external service — if it goes down, flags fall back to alt text and team name.

### Risks

- flagcdn.com outage: handled gracefully since team name is always visible below the flag. Modal remains fully functional.
- ISO code mapping errors: mitigated by defining the mapping once in `lib/copa-teams.ts` and testing all 32 entries.

## Implementation Notes

- Use `w80` size variant (80×60px) for the 64×64 grid cards — crisp on retina screens.
- Add `next/image` `<Image>` wrapper with `width={64} height={48}` and `className="rounded-md object-cover"` for consistent sizing across flags with different aspect ratios.
- Set `alt={team.name}` for accessibility.
- The ISO alpha-2 code for Netherlands (Holanda) is `nl`; for South Korea (Coreia do Sul) is `kr`.
